# OneShot Mod

## English

### Description
OneShot is a Minecraft Forge mod that adds bullet-level mechanics to armor and weapons. Players can obtain and upgrade bullet grades through various means, enhancing their equipment's performance.

### Features
- Bullet level system for armor and weapons
- Quick repair kit item for repairing equipment
- Rarity-based visual integration
- Bullet grade enchantment for armor
- Synchronized bullet level indicators on client

### Installation
1. Install Minecraft Forge for 1.20.1
2. Download the latest mod JAR from releases
3. Place the JAR file in your `mods` folder
4. Launch Minecraft with the Forge profile

### Building from Source
```bash
./gradlew build
```

## 中文

### 简介
OneShot 是一个 Minecraft Forge 模组，为盔甲和武器添加弹等级系统。玩家可以通过各种方式获取和升级弹等级，增强装备性能。

### 功能特性
- 盔甲和武器的弹等级系统
- 快速修复工具包物品，用于修复装备
- 基于稀有度的视觉集成
- 盔甲弹等级附魔
- 客户端同步的弹等级显示

### 安装方法
1. 安装 Minecraft Forge 1.20.1 版本
2. 从 releases 下载最新的 mod JAR 文件
3. 将 JAR 文件放入 `mods` 文件夹
4. 使用 Forge 配置文件启动 Minecraft

### 从源码构建
```bash
./gradlew build
```
